Mxf Log Analyzer
================

Application for analyzing *.mxf files' transfer logs.

Dependencies
------------

* Java 8

Building
--------

In project's directory:

`./gradlew[.bat] build`

Running
-------

Assuming distribution is installed in searchable PATH:

`mxf-log-analyzer[.bat] <input_file_path> <log_directory_path> <output_file_path>`

Developing
----------

In project's directory:

`./gradlew[.bat] run`
